# Stellar Burgers

Космическая бургерная, где можно собрать и заказать бургер своей мечты.

## Функциональность

- 🍔 **Конструктор бургеров** - собирайте бургер из ингредиентов
- 📋 **Лента заказов** - просматривайте все заказы в реальном времени
- 👤 **Личный кабинет** - управляйте профилем и смотрите историю заказов
- 🔐 **Авторизация** - регистрация, вход, восстановление пароля

## Технологии

- React
- TypeScript
- Redux Toolkit
- React Router v6
- WebSocket
- CSS Modules

## Установка и запуск

# Клонировать репозиторий
git clone <url-репозитория>

# Перейти в папку проекта
cd stellar-burgers

# Установить зависимости
npm install

# Создать файл .env
cp .env.example .env

# Запустить проект
npm start
Переменные окружения
Создайте файл .env в корне проекта:
BURGER_API_URL=https://norma.education-services.ru/api

Скрипты
npm start - запуск проекта в режиме разработки

npm run build - сборка проекта

npm test - запуск тестов

npm run lint - проверка кода линтером

npm run lint:fix - исправление ошибок линтера
